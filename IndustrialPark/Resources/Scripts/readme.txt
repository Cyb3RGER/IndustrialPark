* ANIM_asset_importer_gcn.ms and ANIM_asset_importer_xbox.ms:
	ANIM asset importer scripts for 3ds max by Seil and igorseabra4. After importing the rigged model, use one of the scripts to import an ANM file. The GCN script imports GameCube ANM files and the Xbox script imports both Xbox and PS2 ANM files.

* importvcolorobj_v1.3.ms:
	OBJ importer script for 3ds Max originally by Chris Cookson, modded by Shadowth117 to allow importing of OBJ with vertex colors.

To run one of the scripts, in 3ds Max go to the Utilies tab, select 'MAXScript', then 'Run Script' and open one of the scripts.
